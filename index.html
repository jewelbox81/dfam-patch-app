     Cable Colour
  • Resize the app by resizing your browser window or pinch-zoom on mobile and tablet devices.
  • Click/tap a patch point, then click/tap a destination patch point to set a cable. Double-click/tap a cable to remove it.
  • Multiple cables can connect to a single patch point and you can create a 'dead-patch' by double-clicking an active patch point.
  • Click a cable to select it (Shift-click to select multiple), then choose a colour from the picker. Choosing a colour with no cable selected sets the colour for the next cable.
  • Use the small grey circular handles halfway along a cable to adjust its arc. Click and drag to move them.
  • Click and drag control knobs and click toggle switches to mimic the positions on your actual DFAM.
  • Saving the patch will save a .json file with the name given in the 'Patch Name' field directly to your downloads folder. You can then use this file to reload the patch into the app at a later date.
  • In the same manner, exporting a PNG will save a .png snapshot of the current app state, including the patch name and notes, with dimensions: 2560 x 1440.
  • A colour slider for the control position indicators is available to help people who see colours differently. Set it to change the hue. (Not available on Safari due to filter restrictions.)

Enjoy!
